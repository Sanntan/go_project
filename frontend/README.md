# Bank AML System - Frontend Dashboard

Веб-интерфейс для тестирования и демонстрации работы Bank AML System.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение откроется автоматически на `http://localhost:3000`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Предварительные требования

Перед запуском фронтенда убедитесь, что оба Go сервиса запущены:
- Ingestion Service на порту 8080
- Fraud Detection Service на порту 8081

## Функциональность

- ✅ Отправка транзакций через веб-форму
- ✅ Отображение статуса сервисов (Онлайн/Офлайн)
- ✅ Список всех транзакций с их статусами
- ✅ Детальная информация о каждой транзакции
- ✅ Визуальное отображение уровня риска (Низкий/Средний/Высокий)
- ✅ Отображение флагов риска
- ✅ Автоматическое обновление списка транзакций
- ✅ Все поля на русском языке

## Технологии

- Vue.js 3 (Composition API)
- Vite (dev server)
- Axios для HTTP запросов
- Чистый CSS для стилизации
- LocalStorage для хранения истории транзакций
